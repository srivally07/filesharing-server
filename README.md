# Secure File Sharing Server

A secure file sharing backend built using Spring Boot and MongoDB.

## Features

- File upload with encryption (AES)
- Download limit per link
- Expiry control
- Token-based secure links
- Download analytics
- Automatic cleanup scheduler
- File size protection

## Tech Stack

Java
Spring Boot
MongoDB
HTML/CSS/JavaScript

## Example Link

http://localhost:8080/f/{token}
